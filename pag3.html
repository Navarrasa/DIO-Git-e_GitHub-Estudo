Git e GitHub

O que é o Git?

O Git é um Sistema de Controle de Versão Distribuído, criado em 2005 por Linus Torvalds. É utilizado por mais de 90% dos desenvolvedores, sendo o DVCS mais popular e utilizado do mundo. O Git nos proporciona o registro de alterações de um arquivo e permite uma maior coordenação do trabalho de uma equipe em um repositório compartilhado.

Para que é usado o Git?

O Git é utilizado como uma ferramenta de controle de versões de arquivos em uma equipe de desenvolvedores, possui ferramentas que possibilitam a alteração de arquivos do projeto que está sendo desenvolvido, porém não é o arquivo original, o destaque do Git é que cada usuário recebe uma cópia do arquivo localmente para alterar, sem alterar o arquivo principal, proporcionando uma melhor organização e possibilidade de corrigir bugs comparando versões antes das alterações realizadas no arquivo.

O que é GitHub?

O GitHub é uma plataforma de desenvolvimento colaborativo, onde promove a colaboração de mais de 94 milhões de desenvolvedores, que armazenam códigos de diferentes linguagens de programação, que são em sua maioria, públicos, permitindo que os outros acompanhem de perto o seu progresso.

O GitHub também funciona como rede social, conectando desenvolvedores e usuários, podendo colaborar com o projeto dos outros, quanto os outros colaborando com o seu projeto.

Para que é usado o GitHub

O GitHub é utilizado para ajudar os desenvolvedores a armazenar e administrar os arquivos e faz o registro de mudanças. Como usuário, você pode descarregar programas ou aplicativos, e da mesma maneira, pode colaborar com seu desenvolvimento oferecendo melhorias e discutindo as questões que interessam nos fóruns temáticos.
