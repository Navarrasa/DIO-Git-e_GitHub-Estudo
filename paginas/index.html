Vamos aprender a trabalhar com o Git e GitHub!

Após 1950, com a crecente demanda de armazenamento de arquivos e ineficiência dos Sistemas de Gerenciamento de Banco de Dados (SGBD) da época, foram criados diversos softwares que traziam consigo novas funcionalidades, os nomeados Version Control System, ou Sistema de Controle de Versões, que são necessários para controlar as modificações que o arquivo sofre ao longo de seu desenvolvimento.

Esses sistemas utilizados para armazenamento de versões de arquivos, são conhecidos como VCSs, ou Version Control Systens (Sistema de Controle de Versões). O mais conhecido e utilizado por muitas empresas globalmente, é o Git, um DVCS, o qual falaremos no próximo tópico, VCS.

Para aprender mais sobre o Git, clique aqui!
